# FortniteStatistics
Fortnite-Statistics is a stats tracker for the video game Fortnite, directly making calls to Epic Games Fortnite Web API for a specified user's stats and recent matches, and a separate Web API that requests the latest news for the game. The back end consists of an Express.js application. The front end is an Angular 5 web application being hosted as a static web page on an AWS S3 Bucket.

Site URL: https://fortnite-statistics.com
